# AI Patterns

## `agent-adventure`

Example showing a AI Agent powered text-based adventure game. Integrates user input from REST DSL, an AMQP queue for requests to the agent, and a route with `langchain4j-agent` to call an LLM model.

## `camel-quarkus-mcp`

Example showing how an existing Camel route can be called by an AI Agent through a Quarkus MCP server.
